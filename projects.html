Projects

This is a list of the main projects I've worked on:

  • EternalBasher: A collection of bash scripts for DOOM Eternal modding on Linux.
  • EternalModLoaderCpp: A cross-platform C++ rewrite of EternalModLoader.
  • EternalModManager: A cross-platform mod manager for DOOM Eternal, built with Electron.
  • EternalPatcherLinux: A C rewrite of EternalPatcher for Linux.
  • idRehashLinux: A C rewrite of idRehash for Linux.
  • EternalPatchManifestLinux: A C rewrite of the DEternal_patchManifest Python script for Linux.
  • EternalResourceExtractor: A cross-platform C++ utility to extract DOOM Eternal's .resources files.
  • DivinityMachine: A cross-platform C++ texture converter from DDS to DOOM Eternal's TGA format.
  • BlangJsonGeneratorUI: A cross-platform C# GUI utility for creating string mods for DOOM Eternal.
  • DOOMLauncher: A C# launcher for DOOM (2016) to apply devmode bypass patches on startup.
  • SAMUEL: A C++ tool for extracting assets from DOOM Eternal.
  • EternalModLoader: A C# tool to load mods into DOOM Eternal's .resources files.
